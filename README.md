# 山田尖
基于 Spring Boot 搭建的开发框架沿用 Spring Cloud架构 原型

###### 模块说明：
    - stj-api-services API接口jar项目
        - stj-business-api-service 山田尖电商API
        - stj-wechat-api-service 微信API
    - stj-common 公共工具
    - stj-config 配置
    - stj-services web服务
        - stj-business-web 山田尖电商web
        - sjt-oms-web 后台管理平台
